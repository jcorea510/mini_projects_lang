# mini_projects_lang
Bunch of miniproject to improve my knowledge of progamming languages and development frameworks

- Circuitx: An extremely simple simulator of electronic circuits
- Gesture controller: Machine learning app using YOLO models to recognize hand gestures, and using C++.
- Hiragana: Machine Learning app using custom neural network to recognize hiragana.
- HyprKeys: A Rust app to read hyprland config keys to help find them using a simple fuzzy finder.
- rustygame: A simple top-down game in Rust.
- serialprotocols: Simulation in system verilog of UART and SPI serial communication protocol.
- sqlSensors: A python program that connects to a SQL database in mysql and insert data coming from sensor in raspberry pi devices
that communicates by TCP protocol.
- stm32multidevice: Rust embedded program to test async rust feature in STM32 blue pill. It uses PWM, ADC, I2C among others.
- updChat: A chat app in C++ using UDP protocol. Two implementations: P2P and Client-Server-Client.
- wavescan: A reimplementation in Rust of AM single side band simulator using Ratatui (modern TUI library for Rust).
